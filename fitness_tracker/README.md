# FitTrack Pro

Offline-First Fitness Tracker App

## Features

- Activity Tracking
- Step Counter
- Calories Tracking
- Workout History
- Statistics Dashboard
- Offline SQLite Database
- Clean Architecture
- Riverpod State Management

## Tech Stack

- Flutter
- Riverpod
- Drift (SQLite)
- GoRouter
